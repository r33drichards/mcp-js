# Quick Start Overview

Pick the entry point you want:

- [Claude Code](claude-code.md)
- [Codex](codex.md)
- [Cursor](cursor.md)
- [Generic MCP](generic-mcp.md)
- [curl](curl.md)
- [Bundled CLI](bundled-cli.md)

If you do not already have `mcp-v8` installed, start with:

```bash
curl -fsSL https://raw.githubusercontent.com/r33drichards/mcp-js/main/install.sh | sudo bash
```

That is the same GitHub install script used in
[Install the Server](../how-to/install-server.md).
