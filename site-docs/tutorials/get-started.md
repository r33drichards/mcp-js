# Get Started with MCP-v8

This tutorial will guide you through setting up your first MCP server and running a simple example.

## What You'll Build

By the end of this tutorial, you'll have:
- A working MCP server instance
- A basic understanding of the MCP protocol
- The ability to execute JavaScript functions via MCP

## Prerequisites

Before starting, ensure you have:
- Node.js version 16 or higher installed
- Basic knowledge of JavaScript/TypeScript
- A terminal/shell environment
